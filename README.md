# Juego de Arkanoid

Juego de Arkanoid/Breakout en HTML, CSS y JavaScript con cero dependencias. Se abre directamente en el navegador sin necesidad de build ni servidor.

## Cómo jugar

Abre `index.html` en el navegador o ejecuta:

```
open index.html
```

## Características

- Pala controlada con el ratón o las teclas de flecha
- Bola con física de rebote en paredes, techo, pala y bloques
- 7 colores de bloques con animación de explosión al destruirse
- Sonidos de rebote y rotura (con botón de mute)
- 3 niveles con velocidad de bola creciente
- Contador de vidas y puntuación

## Tecnología

- HTML5 Canvas para el renderizado
- Spritesheet en `assets/spritesheet-breakout.png`
- Sonidos en `assets/sounds/`
- Sin dependencias externas
